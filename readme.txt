CREAR PROYECTO 32BITS CON VUE CLI
------------------------
INSTALAR VUE ROUTER Y VUEX
------------------------
IMPLEMENTAR VUE ROUTER Y VUEX COMO PLUGIN
- CREAR ROUTER.JS
    - CREAR RUTAS PARA VISTAS DE: 
        INCIO: RUTA /
        BUSQUEDA DE PRODUCTOS: RUTA /BUSQUEDA
        VENTAS DE PRODUCTOS: RUTA /VENTAS
        TOTAL DE VENTAS: /TOTAL
        PAGINA 404: *
    - CARGAR RUTAS CON LAZY LOADING
- CREAR STORE.JS
    - CREAR ESTADOS:
        - TITULO
        - SUBTITULO
        - MENU
        - LISTADO DE PRODUCTOS DISPONIBLES
            - CODIGO, NOMBRE, STOCK, PRECIO
        - LISTADO DE PRODUCTOS VENDIDOS
            - CODIGO, NOMBRE, PRECIO
------------------------

SISTEMA DE VENTAS 32BITS
------------------------
VISTA RAIZ (APP.VUE)
- CREAR MENU PARA NAVEGAR ENTRE VISTAS (ROUTER-LINK)
- ZONA PARA DESPLEGAR VISTAS (COMPONENTES CON ROUTER-VIEW CON TRANSITION)
    - USAR ESTE CSS PARA LA TRANSITION
        .vista-enter-active, .vista-leave-active {
            transition: opacity .1s;
        }
        .vista-enter, .vista-leave-to{
            opacity: 0;
        }
------------------------
PAGINA DE PRESENTACION (INICIO.VUE)
-H1 CON EL TITULO (ESTADO TITULO)
-H2 CON UN SUBTITULO (ESTADO SUBTITULO)
* LOS ESTADOS CARGARLOS CON MAPSTATE*
------------------------
PAGINA BUSQUEDA DE PRODUCTOS
-TXTBOX PARA FILTRAR POR CODIGO. SI NO INGRESA, DEVUELVE TOTAL
-MOSTRAR LA CANTIDAD DE PRODUCTOS TOTALES (Cantidad de juegos disponibles: N)
    - UTILIZAR EL FONDO DE COLOR SUGERIDO
    - EL LISTADO DEBE ESTAR EN UN COMPONENTE DISTINTO QUE DEBE SER IMPORTADO
-LISTAR (V-FOR) DE ACUERDO A FILTRO: CODIGO | NOMBRE | STOCK | PRECIO
------------------------
PAGINA VENTA DE PRODUCTOS
- MOSTRAR LA CANTIDAD DE PRODUCTOS CON STOCK
- LISTAR SOLO LOS QUE TIENEN STOCK, LOS PRODUCTOS DESTACADOS DEBEN LLEVAR NEGRITA
- CODIGO | NOMBRE | STOCK | PRECIO | BOTON VENDER
- CREAR FUNCION PARA VENDER UN PRODUCTO:
    - PROCESAR VENTA: 
        -   DESCONTAR 1 DEL STOCK. DEBE TARDAR 2 SEGUNDOS (UTILIZAR SETTIMEOUT PARA ESTO)
    - REGISTRAR LA VENTA:
        -   AGREGAR AL OBJETO DE VENTAS EL CODIGO, NOMBRE Y PRECIO DE VENTA. DEBE DURAR 1 SEGUNDO.
    - AL FINALIZAR ESTAS OPERACIONES, MOSTRAR UN MENSAJE ALERTANDO AL USUARIO Q LA VENTA SE EFECTUO
------------------------
TOTAL DE VENTAS REALIZADAS
- LISTAR PRODUCTOS VENDIDOS Y LA CANTIDAD DE ESTOS (UTILIZAR GETTER)
    - DE NO HABER VENTAS, MOSTRAR UN TEXTO INDICANDOLO
- MOSTRAR EL TOTAL DE VENTAS CALCULADO DEL LISTADO ANTERIOR
------------------------
